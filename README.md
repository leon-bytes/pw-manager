# Simple Password Manager

A simple C++ console application that stores up to 5 username and password entries.

## Features

- Add an entry
- List all entries
- Search for an entry by username
- Update a password by username
- Simple numbered menu

## Concepts Used

- Classes
- Private data
- Constructors
- Getters and setters
- Arrays
- Functions
- Loops
- Conditionals

## How to Compile

```bash
g++ main.cpp -o main
```

## How to Run

```bash
./main
```
